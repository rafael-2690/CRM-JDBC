# CRM JDBC

Projeto desenvolvido em Java com o objetivo de praticar a integração entre aplicações Java e bancos de dados relacionais utilizando JDBC.

## Sobre o projeto

Este projeto simula funcionalidades comuns de um sistema CRM (Customer Relationship Management), permitindo a manipulação de dados por meio de operações de cadastro, consulta, atualização e remoção de registros.

O principal objetivo foi aplicar conceitos de persistência de dados, organização de código e boas práticas de desenvolvimento utilizando Java.

## Tecnologias utilizadas

* Java
* JDBC (Java Database Connectivity)
* Maven
* MySQL
* IntelliJ IDEA
* Git e GitHub

## Conceitos praticados

* Conexão com banco de dados
* CRUD (Create, Read, Update, Delete)
* Manipulação de SQL em Java
* Separação de responsabilidades
* Estruturação de projetos Maven
* Tratamento de exceções
* Persistência de dados

## Estrutura do projeto

